Motivace¶

  1. Python „baví“

  2. Python se „neučí“, Python se „píše“

  3. Python je dnes asi nejvíce „geopozitivní“ programovací jazyk

    s množstvím knihoven a nástrojů (GDAL, PROJ, Shapely, Fiona, Rasterio, MapServer mapscript, GeoServer gsconfig, OWSLib, PyWPS, pycsw, …)

    a širokou podporou v deskopových prostředích (GRASS GIS - pygrass, QGIS - pyqgis, Esri ArcGIS - arcpy, …)

Poznámka k datové sadě GISMentors

Data ke školení je stažitelná jako zip archiv (166 MB). Dále můžete využít data ze školení QGIS pro začátečníky - zip archiv (614 MB). Datová sada GISMentors je založena na datech pocházejících pouze z otevřených či veřejných zdrojů jako je EU-DEM, RÚIAN, OpenStreetMap, Dibavod, AOPK a IPR.
