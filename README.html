Pracovní prostředí¶

  • OSGeo4W

  • Data archive: https://gismentors.github.io/pygis/pygis-workshop.zip

  • Jednotlivé kroky https://github.com/gismentors/pygis/tree/master/kurz

    python3 -m pytest tests
